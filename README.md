# rust-solar
A simple solar tracer app re-written in rust.

# Running

```
cargo run
```
# How to use.
 - Select a COM port from the initial list.
 - Once the app is running, you can use the mouse to click LOAD on or off.
 - The display will update once per second.
 - To quit press q.
